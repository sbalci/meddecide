# Calculate Expected Value of Perfect Information (EVPI)

Computes EVPI for value of information analysis.

## Usage

``` r
calculateEVPI(psaResults, willingnessToPay)
```

## Arguments

- psaResults:

  Data frame with PSA results containing cost, utility, strategy

- willingnessToPay:

  Numeric willingness-to-pay threshold

## Value

List with EVPI calculations
