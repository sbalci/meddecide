# Validate Decision Analysis Inputs

Comprehensive validation of user inputs for decision analysis.

## Usage

``` r
validateDecisionAnalysisInputs(data, treeType, options)
```

## Arguments

- data:

  Data frame with analysis data

- treeType:

  Character string specifying analysis type

- options:

  List of analysis options

## Value

List with validation results (valid = TRUE/FALSE, messages = character
vector)
